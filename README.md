## Deployment to GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, set **Source** to "Deploy from a branch".
4. Select the `main` branch and the `/ (root)` folder.
5. Click **Save**.
